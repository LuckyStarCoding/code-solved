Given an unsorted array of integers numbers, determine the length of the longest sequence where the integers appear consecutively, 
meaning the numbers follow one another without any gaps.

<!-- truncate -->

* longestConsecutiveNumberSeq([5,1,-4]) ➞ 1

<TabItem value="solution" label="Solution" >
```jsx 
function longestConsecutiveNumberSeq(nums) {
  let s = Array.from(new Set(nums.sort((a,b)=>a-b)));
    let currentCount = 1;
    let maxCount = 1;
    for (let i = 0; i < s.length; i++) {
        if(s[i + 1] === s[i] + 1) {currentCount++} 
        else {
            maxCount = Math.max(maxCount, currentCount)
            currentCount = 1;
        } 
    }
    return maxCount;
}

```
