Given an array of integers numbers and a number k, find the k most frequent numbers in the array. Here, k represents the number of elements that should be returned, which are the ones that appear the most frequently. 
The order of the result does not matter.

<TabItem value="solution" label="Solution" >
```jsx 
function mostCommonElements(nums, k) {
    let d = new Set(nums);
    let f = [];    
      for (let i of d) {
          f.push([i, nums.filter(el=>el===i).length])
      }
        return f.sort((a,b)=>b[1]-a[1]).map(el=>el[0]).slice(0, k);
    }

```
