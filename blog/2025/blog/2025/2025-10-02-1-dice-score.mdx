Greed is a dice game played with five six-sided dice. Your mission is to score a throw according to these rules:
```
 Three 1's => 1000 points
 Three 6's =>  600 points
 Three 5's =>  500 points
 Three 4's =>  400 points
 Three 3's =>  300 points
 Three 2's =>  200 points
 One   1   =>  100 points
 One   5   =>   50 points
 ```
See the below examples for a better understanding:

| Throw | Score | Explanation |
| :---: | :---: | :--- |
| **5 1 3 4 1** | **250** | $50$ (for the $5$) + $2 \times 100$ (for the $1$s) |
| **1 1 1 3 1** | **1100** | $1000$ (for three $1$s) + $100$ (for the other $1$) |
| **2 4 4 5 4** | **450** | $400$ (for three $4$s) + $50$ (for the $5$) |

Create a function that takes an array of five six-sided throw values and returns the final calculated dice score.

<TabItem value="solution" label="Solution" >
```jsx 
function diceScore(t) {
let obj = {};
let c = 0    
let s = Array.from(new Set(t));
   for (let i of s) {
       obj[i]=t.filter(el=>el===i).length
   }
   for (let i in obj) {
    i==='1' && obj[i] == 3 ? c+= 1000 :
    i==='1' && obj[i] >= 3 ? c+= 1100 :
    i==='6' && obj[i] == 3 ? c+= 600 :
    i==='5' && obj[i] == 3 ? c+= 500 :
    i==='4' && obj[i] == 3 ? c+= 400 :  
    i==='3' && obj[i] >= 3 ? c+= 300 : 
    i==='2' && obj[i] == 3 ? c+= 200 :
    i==='1' ? c+=obj[i]* 100 :
    i==='5' ? c+= 50 :  c;

          }
return c;
   }  
```
