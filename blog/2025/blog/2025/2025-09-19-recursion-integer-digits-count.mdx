    For this task I have incorrectly used the global variable. Now I understand that I can pass the counter as the function argument.
</TabItem>
<TabItem value="solution" label="Solution" >
```jsx 
function count(n, c=1) {
  if (n<0) {n=Math.abs(n)}
  // turns negative into positive
  if (n<10) {return c} 
  else { return count(Math.floor(n/10), c+1)}
  return c;
}
```
